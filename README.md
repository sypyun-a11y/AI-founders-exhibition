# AI 파운더스 1기 · 데모데이 전시회

비전공자 수강생들이 직접 만든 AI 서비스 전시회 (정적 사이트).

## 구성
- `index.html` — 전시 목록 페이지 (히어로 + 카드 그리드 + 검색/득표순 + 페이지네이션)
- `detail.html?s=<이름>` — 서비스 상세 페이지
- `data.js` — 서비스 데이터 + 커버 설정 (여기 한 곳만 고치면 목록·상세 모두 반영)
- `shots/` — 서비스 스크린샷

## 투표
- 로그인 없음. 브라우저별 식별자로 **작품당 1표(중복 차단)**, 여러 작품 투표 허용.
- 표는 Supabase `votes` 테이블에 공유 저장. 켜는 법: `votes-setup.sql` 을 Supabase SQL Editor 에서 1회 실행.
- 미설정 시 localStorage 폴백으로 화면은 정상 동작.

## 배포
빌드 없는 정적 사이트 — Vercel에서 레포를 import 하면 그대로 배포됩니다.
