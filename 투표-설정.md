# 전시회 투표 설정 (로그인 없음)

투표는 **로그인 없이** 동작해요. 브라우저별 익명 식별자(voter)로 **작품당 1표(중복 차단)**, **여러 작품 투표는 허용**.

## 켜는 법 — SQL 1번만 실행
Supabase 대시보드 → **SQL Editor** → [supabase/votes-setup.sql](../supabase/votes-setup.sql) 내용 붙여넣고 **Run**.

→ 끝. 표는 Supabase `votes` 테이블에 모두에게 공유 저장돼요.

## 동작
- 카드/상세의 ❤️ → 투표 (같은 작품 중복 ❌, 여러 작품 ⭕)
- SQL 실행 전이라도 화면은 정상 동작 (이때는 브라우저 localStorage에만 임시 저장)

> 참고: 로그인 방식(구글 OAuth)과 IP 기반 Edge Function(`supabase/functions/vote`)은 이제 안 써요. 무시하거나 삭제해도 됩니다.
> 한계: 로그인이 없어서 시크릿창·캐시삭제로 중복 우회는 가능해요. "응원·좋아요" 수준의 투표에 적합합니다.
